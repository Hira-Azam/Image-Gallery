**Image-Gallery**
This is a responsive Image Gallery web application built with HTML, CSS, and JavaScript. It allows users to browse through categorized images with filtering and search functionality, providing an intuitive and visually appealing interface.

**Features**
**Category Filters:** Easily filter images by categories such as Flowers, Laptops, Bags, Sneakers, and Watches.
**Search Functionality:** Search images by name to quickly find specific items.
**Image Preview:** Click on any image to open a larger preview with navigation arrows to browse images within the preview mode.
**Responsive Design:** The layout adapts smoothly across different screen sizes, from desktop to mobile devices.
**Clean and Animated UI:** Includes hover effects, animated headings, and interactive elements for a better user experience.

**Technologies Used**
HTML5 for page structure.
CSS3 for styling and animations.
Vanilla JavaScript for interactive filtering, searching, and image preview functionality.

**Usage**
Open the gallery.html file in a modern web browser.
Use the category buttons to filter images by category.
Use the search input to filter images by name.
Click on any image to view it in an enlarged preview box, navigate between images using arrows, and close the preview with the close button.
